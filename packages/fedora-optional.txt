# Aplicativos opcionais (Fedora) — menu de seleção múltipla.
# Equivalente ao packages/arch-optional.txt, com os nomes/origens do Fedora.
#
# Formato: <entrada> | <rótulo exibido no menu>
#   nome                        → pacote nativo do dnf
#   flatpak:<app-id>            → Flatpak do Flathub (sem pacote nativo no Fedora)
#   copr:<dono>/<projeto>:<pkg> → habilita o COPR e instala
#   curl:<url>                  → instalador oficial do próprio programa
#
# Linhas em branco e comentários (#) são ignorados pelo instalador.
