# Navegadores (Fedora) — menu de seleção múltipla.
# Equivalente ao packages/arch-browsers.txt.
#
# DECISÃO DE PROJETO: navegadores NÃO são instalados via Flatpak aqui, por
# preferência explícita do autor. Cada um usa a origem nativa recomendada pelo
# próprio projeto — COPR, repositório oficial do fornecedor ou instalador
# oficial. Todos os endereços abaixo foram verificados no Fedora 44.
#
# Formato: <entrada> | <rótulo exibido no menu>
#   nome                        → pacote nativo do dnf
#   copr:<dono>/<projeto>:<pkg> → habilita o COPR e instala
#   repo:<pacote>@<url-.repo>   → adiciona o repositório do fornecedor e instala
#   curl:<url>                  → instalador oficial do próprio programa

firefox | Firefox — nativo do Fedora, padrão

copr:guillermodotn/zen-browser:zen-browser | Zen Browser — baseado no Firefox, foco em produtividade [COPR]

copr:imput/helium:helium-bin | Helium Browser — Chromium minimalista e privado [COPR]

# O Brave NÃO está nos repositórios oficiais do Fedora. O instalador oficial
# adiciona o repositório da própria Brave (brave-browser-rpm-release.s3.brave.com),
# que é o caminho recomendado por eles e o que mantém o navegador atualizado.
curl:https://dl.brave.com/install.sh | Brave — Chromium com bloqueio de anúncios integrado

repo:librewolf@https://rpm.librewolf.net/librewolf.repo | LibreWolf — Firefox focado em privacidade [repo oficial]

# Repositório oficial do Vivaldi, com verificação GPG. Preferido a baixar um
# .rpm avulso: o RPM avulso congela a versão e nunca recebe atualização.
repo:vivaldi-stable@https://repo.vivaldi.com/archive/vivaldi-fedora.repo | Vivaldi — navegador altamente personalizável [repo oficial]
